# Advanced Rails

## 動作環境

必須
- RealTrainMod 1.12.2(NGT) or 1.7.10(KaizPatch) 
- RTM Wire51 Pack

推奨
- RAM 16GB以上

## 導入

0. Kozu_RE_XまたはBetter_Rail_Electronicsをmodsファイルから削除（すでに導入されている場合）
1. Advanced_Rails.zipをmodsフォルダにコピー
2. Minecraftを起動し、各モデルが追加されていることを確認
